# Arkanus

Jogo web/PWA de estratégia hermética por turnos.

## Instalação

npm install

## Desenvolvimento

npm run dev

## Build

npm run build

## Deploy Firebase

firebase deploy

## Configuração

O arquivo `firebase-applet-config.json` inicializa o Firebase no MVP atual.
Na ausência deste, preencha as variáveis de ambiente baseadas no seu `.env.example`.
